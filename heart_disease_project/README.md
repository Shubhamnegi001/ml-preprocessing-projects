# ❤️ Heart Disease Data Preprocessing

## 📌 Overview

This project focuses on preparing a heart disease dataset for machine learning.

---

## 🔍 Steps Performed

### 1. Data Cleaning

* Replaced invalid values (0) in:

  * Cholesterol
  * Resting Blood Pressure

### 2. Exploratory Data Analysis

* Distribution plots
* Count plots with target variable
* Boxplots & violin plots

### 3. Encoding

* One-hot encoding for categorical variables

### 4. Feature Scaling

* StandardScaler applied to numerical features

---

## 📊 Dataset Info

* Rows: 918
* Features: 12

---

## 🧠 Key Insight

Proper handling of invalid medical values is critical for reliable predictions.
